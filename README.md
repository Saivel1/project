# project

Для работы файла main.py понадобяться: 

Python 3.12.4
https://www.python.org/downloads/

pip 24.3.1
```
python.exe -m pip install --upgrade pip
```
Создать виртуально пространство 
```
python -m venv .venv  
```
Запустить его 

```
.venv\Scripts\activate
```

Установить библиотеку aiogram

```
pip install aiogram    
```
